# Project 1: Explore Weather Trends

## Data 
The dataset is provided in 3 csv files that was extracted from Udacity using SQL queries: [city_data](https://github.com/jmt7080/Data_Analyst_Nanodegree/blob/main/Project%201%20Explore%20Weather%20Trends/city_data.csv), [city_list](https://github.com/jmt7080/Data_Analyst_Nanodegree/blob/main/Project%201%20Explore%20Weather%20Trends/city_list.csv), and [global_data](https://github.com/jmt7080/Data_Analyst_Nanodegree/blob/main/Project%201%20Explore%20Weather%20Trends/global_data.csv). 

## Objectives 
The goal of this project is to use [Python with Jupyter notebook](https://github.com/jmt7080/Data_Analyst_Nanodegree/blob/main/Project%201%20Explore%20Weather%20Trends/Weather%20Trends%20Notebook.ipynb) to merge and clean the 3 datasets, create visualizations, then explain the findings in a [presentation](https://github.com/jmt7080/Data_Analyst_Nanodegree/blob/main/Project%201%20Explore%20Weather%20Trends/report.pdf).
